问：es6 之前的 js 里，除了函数这种结构以外，还有其他的结构能生成作用域吗？

在 es6 之前，js 就只有基于函数的作用域，其他结构都不会创建作用域

函数作用域就是，属于这个函数的全部变量都可以在整个函数的范围内使用以及复用

隐藏数据和变量到函数作用域里，其实基于的是一种最小授权原则，或者最小暴露原则，简单来说就是在全局中暴露更少的变量

如果所有的变量都在全局中，当然可以在所有的内部嵌套作用域中访问到它们，但是这样会暴露更多的变量或者函数，这些本应私有

问：聊下 IIFE 和 NFE？

IIFE 全称 Immediately Invoked Function Expression => 立即执行函数

NFE 全称 Named Function Expression, => 具名的函数表达式

这里就会引出一个问题，如何区分一条语句是函数声明还是函数表达式？

如果 function 关键字是语句中的第一个词，那么就是函数声明语句，否则就是函数表达式
