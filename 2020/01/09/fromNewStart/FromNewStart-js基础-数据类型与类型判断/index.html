一、前言

本文主要介绍的内容，为 javascript 这门语言的数据类型与进行类型判断的几种方式

本文参考的文档如下：

  • ecmascript中文文档

二、数据类型

大体可以分为两类：

  • 基本类型
  • 引用类型

我们可以简单的将基本类型的值分为：

  • 值为有限个数
    • Undefined
    • Null
    • Boolean
  • 值为无限个数
    • String
    • Symbol
    • Number
    • BigInt

引用类型看大类的话，只有一个 Object

但是我们常见的，Function, Array, Data, Regexp 等构造函数的实例或者字面量，都是对象，他们的原型链找到最上层，都可以找得到 Object.prototype
