前言

技术日新月异，随着 MVVM view 层框架、nodejs 的普及，jser 能做的事情越来越多，所负责的事情也越来越多。其中需要进行大量的数据处理。数组便是其中最常用的数据结构，但很多同学不清楚一些数组的方法如何使用，或者只知道它本身的最基本的能力，缺乏对其的执行过程的了解。

通过阅读本文，你将会学到如何使用位于 Array.prototype 上的 push 方法，了解到他们的执行过程以及一些技巧。对一个技术细节理解很透彻，也容易引起面试官的兴趣~

在 Vue 原来越普及的今天，我们更习惯于去使用其 VM 的特性，做的很多工作实际就是在数据绑定、处理数据。

push 函数的具体执行过程

我们描述的主体为 push 函数，该函数引用的访问方式为： Array.prototype.push， 首先来看下基本信息：

  1. 参数：零个或多个
  2. 作用：将参数顺序的追加到调用者末位
  3. 返回：追加结束后，调用者的 length 值
